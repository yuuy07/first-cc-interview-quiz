# llm-wiki-interview

本目录为**合并后的单一技能**：`SKILL.md` 说明 Raw 层与 Wiki 层如何衔接；细则拆在 `references/`。

- **`references/raw-layer.md`**：来自 Obsidian 稿 `LLM-WIKI-INTERVIEW-BUILD-RAW-SKILL.md`，只管 `raw/`。
- **`references/wiki-layer.md`**：来自 Obsidian 稿 `LLM-WIKI-INTERVIEW-BUILD-SKILL.md`，只管从 `raw/` 编译到 `wiki/`。

整体模式参考 Andrej Karpathy 的 [llm-wiki gist](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)（**不在此仓库复述或翻译该 gist 正文**；需要时直接打开链接阅读原文）。
